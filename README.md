docker-builder
==============

docker builder for d2hub

d2hub의 자동빌드기능에서 사용하는 docker image builder.
d2hub에서 요청을 받아서 이미지를 빌드한 후 결과를 다시 d2hub쪽으로 보낸다.

go version : 1.10.2
